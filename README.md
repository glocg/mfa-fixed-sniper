# MFA Fixed Vanity Sniper

# Usage
1) Install Node.js and Golang.
2) Install `ws` and `extract-json-from-string ` modules.
3) Run get_mfa.bat.
4) Wait until the MFA token is taken.
5) Run the sniper.bat file when the MFA token is taken.
